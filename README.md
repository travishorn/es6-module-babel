# ES6 Modules with Babel

Use ES6 modules and transpile them to ES5 for use in browsers.

## Usage

Run `gulp build` to build the `src` directory and output to the `dist` directory.

Optionally, run `gulp watch` to watch continuously watch the `src` directory and build when something changes.
